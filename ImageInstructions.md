# Website Image Replacement Guide

Hello! To replace the placeholder images on your website with your own professional photos, please follow this simplified guide.

**All images should be placed directly inside a single `/images/` folder in your project's `public` directory.**

We recommend optimizing your images for the web to ensure fast loading times.

---

## Simplified Folder Structure

```
public/
└── images/
    ├── hero-home.jpg
    ├── hero-about.jpg
    ├── room-standard.jpg
    ├── gallery-main-1.jpg
    ├── ... and so on
```

---

## Image Details & File Names

Large, high-quality images for the top section of each page.
- `hero-home.jpg` (Homepage)
- `hero-about.jpg` (About Us page)

- `hero-locations.jpg` (Locations page)
